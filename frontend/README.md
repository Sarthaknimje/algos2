# DriveFi Frontend

React + Vite + TailwindCSS frontend for the DriveFi Traffic-to-Earn application.

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The application will start at `http://localhost:5173`

### Build

```bash
npm run build
```
